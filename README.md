Project Title
Will You Be My Girl – Interactive Proposal Website
Description
This project is a small romantic web page created to ask a special person a proposal question in a creative way. The website displays a cute interface where the user clicks an envelope to reveal a message asking “Will you be my girl?” and then responds by choosing Yes or No.
The project focuses on a simple, visually appealing experience using basic web technologies.
Features
Interactive envelope animation
Romantic proposal message
Yes / No response buttons
Lightweight static website
Mobile-friendly design
Technologies Used
HTML
CSS
JavaScript
How It Works
The website loads a romantic themed interface.
The user is asked to click an envelope.
When clicked, the proposal message appears.
The user can choose Yes or No as the response.
